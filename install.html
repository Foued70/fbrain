Dependencies

Baby Brain Toolkit (BTK) depends on CMake, ITK, VTK, Tclap library and OpenMP.

Install

Download the sources here

Create a directory 'fbrain-build' 

$ mkdir fbrain-build
And then compile: 

$ cd fbrain-build 
$ cmake ../fbrain 
$ make
For Further informations, here is the documentation (PDF), and here is a link to the QuickStart wiki page.
